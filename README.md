# 📚 Book Store

A full-stack Book Store application with authentication, dark mode, and a responsive UI.

## 🚀 Features
- 📖 Browse books
- 🔍 Search functionality
- 🌙 Dark mode toggle
- 🔑 User authentication
## 🛠️ Tech Stack
**Frontend:** React.js, TailwindCSS  
**Backend:** Node.js, Express.js, MongoDB
